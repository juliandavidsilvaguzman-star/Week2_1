# Week2_1

# Conceptos clave
Entradas (X): variables que representan las características del problema
Pesos (w): importancia de cada entrada
Bias (b): ajusta el umbral de decisión
Puntaje (z): combinación lineal de entradas y pesos
Función de activación: transforma el puntaje en una salida binaria (0 o 1)


# Descripción del código
El programa realiza los siguientes pasos:
1. Define las entradas, pesos iniciales y bias
2. Calcula el puntaje z mediante el producto punto
3. Aplica una función de activación basada en umbral
4. Evalúa el perceptrón para todas las combinaciones de entrada
5. Muestra los resultados (entrada, puntaje y salida)
